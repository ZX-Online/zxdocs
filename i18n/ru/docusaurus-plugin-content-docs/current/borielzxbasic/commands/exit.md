# EXIT

## Синтаксис

```
EXIT DO
EXIT FOR
EXIT WHILE
```

## Описание

Немедленно прерывает текущий цикл и переносит выполнение за соответствующую закрывающую цикл инструкцию: **LOOP**, **NEXT** или **END WHILE**.

## Примечание

* Эта функция не совместима с Sinclair ZX Basic
